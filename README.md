# Лабораторная работа 2: Redux Calculator

## Описание проекта

Проект представляет собой калькулятор на React с использованием Redux для управления состоянием.

## Структура проекта

```
Lab2/
├── myapp/                    # Основное приложение
│   ├── src/
│   │   ├── redux/
│   │   │   ├── types.js              # Типы экшенов
│   │   │   ├── action.js             # Action creators
│   │   │   └── reducers/
│   │   │       ├── calculatorReducer.js    # Редьюсер калькулятора
│   │   │       └── reducer.js              # Root reducer
│   │   ├── pages/
│   │   │   └── CalculatorPage.js     # Главная страница
│   │   ├── App.js
│   │   ├── App.css
│   │   └── index.js
│   ├── public/
│   ├── package.json
│   └── README.md
```

## Требования к работе

✅ Создано поле input type="number" и две кнопки (add, clear)

✅ Первый раз кликая на add получаем в списке число с поля инпут

✅ В последующие разы кликая на add в списке появляется сумма предыдущего числа и нового

✅ Кликая на clear список полностью очищается

✅ Реализовано 4 калькулятора: сложение, вычитание, умножение, деление

✅ Вся логика разделена по правильной структуре Redux:
   - Отдельно созданы типы (types.js)
   - Отдельно созданы акшины (action.js)  
   - Отдельно созданы редусеры (calculatorReducer.js и reducer.js)

## Как запустить

```bash
cd myapp
npm start
```

Приложение откроется на http://localhost:3000

## Технологии

- React 19.1.1
- Redux 5.0.1
- React-Redux 9.2.0

## Архитектура Redux

Проект строго следует архитектуре Redux:

1. **types.js** - содержит константы типов экшенов
2. **action.js** - функции-создатели экшенов (action creators)
3. **calculatorReducer.js** - обработка логики состояния
4. **reducer.js** - комбинирование редьюсеров через combineReducers
