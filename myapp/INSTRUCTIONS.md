# Инструкция по запуску

## Папка приложения

```
C:\Project React Redux\Lab2\myapp
```

## Как запустить

1. Откройте папку **myapp** в VSCode как корневую папку проекта
2. Или используйте Terminal в VSCode и выполните:

```bash
cd C:\Project React Redux\Lab2\myapp
npm start
```

## Что должно открыться

- Только **Calculator** с 4 калькуляторами (Addition, Subtraction, Multiplication, Division)
-- Должен открыться только Calculator

## Проверка

Если видите что-то отличное от страницы калькулятора, проверьте, что вы находитесь в папке `myapp` и запустили команду из неё.
