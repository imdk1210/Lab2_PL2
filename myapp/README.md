# Calculator with Redux

Это React приложение с использованием Redux для управления состоянием калькулятора.

## Структура проекта

```
myapp/
├── src/
│   ├── redux/
│   │   ├── types.js                    # Типы экшенов
│   │   ├── action.js                   # Экшены (действия)
│   │   └── reducers/
│   │       ├── calculatorReducer.js    # Редьюсер для калькулятора
│   │       └── reducer.js              # Главный редьюсер (rootReducer)
│   ├── pages/
│   │   └── CalculatorPage.js           # Главная страница с калькулятором
│   ├── App.js                          # Корневой компонент
│   ├── App.css                         # Стили
│   └── index.js                        # Точка входа
├── public/                             # Статические файлы
└── package.json                        # Зависимости проекта
```

## Функционал

Приложение предоставляет 4 независимых калькулятора:
1. **Addition (+)** - Сложение
2. **Subtraction (-)** - Вычитание
3. **Multiplication (*)** - Умножение
4. **Division (/)** - Деление

### Логика работы:

1. Вводите число в поле input (type="number")
2. При первом нажатии на кнопку "Add" число добавляется в список
3. При последующих нажатиях выполняется операция между предыдущим результатом и новым числом
4. Кнопка "Clear" очищает список соответствующего калькулятора

## Запуск проекта

```bash
cd myapp
npm install  # Установить зависимости (если еще не установлены)
npm start    # Запустить проект
```

Проект откроется в браузере по адресу: http://localhost:3000

## Правильная структура Redux

Проект следует правильной архитектуре Redux:

1. **types.js** - определяет константы типов экшенов
2. **action.js** - создает экшены (action creators)
3. **calculatorReducer.js** - обрабатывает логику состояния
4. **reducer.js** - комбинирует все редьюсеры
